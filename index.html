      • Как добавить конфигурацию в ZET9:
        1. Скачайте файл конфигурации на компьютер
        2. Найдите скачанный файл (обычно в папке "Загрузки")
        3. Откройте проводник Windows (жёлтая папка на панели задач)
        4. Перейдите по пути в адресную строку проводника:
          C:\ZET9\LANGUAGE\Russian\Config
        5. Нажмите Enter - откроется нужная папка
        6. Скопируйте скачанный файл конфигурации (Ctrl+C)
        7. Вставьте его в открытую папку (Ctrl+V)
        8. При необходимости подтвердите замену файла
        Совет: Если папки ZET9 нет на диске C:\, ищите её там, куда вы устанавливали программу.
      • Школа Аструма Конфигурация Astrooma_Horar_Light
      • Школа ПроАстро Конфигурация ШколаПроАстро
